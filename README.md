1. About
SecurePassValut is a robust, lightweight password manager designed to securely store and manage your sensitive credentials. Built with a focus on simplicity and security, it ensures your passwords remain encrypted and readily accessible only to you.

2. Features
The current core features of SecurePassValut include:

Secure Storage: Acts as a centralized password vault.

High-Level Encryption: All stored data is secured using strong cryptographic methods.

Simple Management: Easy-to-use interface for adding, retrieving, and managing passwords.
